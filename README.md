 # Netflix User Analytics

## Project Overview
This project analyzes Netflix user behavior and applies Machine Learning algorithms to predict subscription renewal and monthly spending.

## Dataset Features
- Age
- Gender
- Subscription Type
- Watch Hours Per Week
- Devices Used
- Favorite Genre
- Ad Clicks
- Monthly Spend
- Subscription Renewed

## Algorithms Used
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)
- Linear Regression

## Objectives
- Predict subscription renewal
- Analyze user behavior
- Predict monthly spending
